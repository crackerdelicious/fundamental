สารบัญ
  1. ปัจจัยแวดล้อมของการลงทุนและตลาดการเงิน
  2. ตราสารการเงินและการตัดสินใจลงทุน
  3. อัตราผลตอบแทนและความเสี่ยงจากการลงทุนในหลักทรัพย์
  4. อัตราผลตอบแทนและความเสี่ยงของกลุ่มหลักทรัพย์
  5. การวิเคราะห์เศรษฐกิจและอุตสาหกรรม
  6. การวิเคราะห์บริษัท
  7. การวิเคราะห์หลักทรัพย์โดยใช้ปัจจัยทางเทคนิค
  8. การลงทุนในหลักทรัพย์ต่างประเทศ NPL

สารบัญ

  • บทที่ 1 ปัจจัยแวดล้อมของการลงทุนและตลาดการเงิน

    1. ปัจจัยแวดล้อมการลงทุน
    2. ตลาดการเงิน
    3. บทสรุป

  • บทที่ 2 ตราสารการเงินและการตัดสินใจลงทุน

    1. ลักษณะและประเภทของตราสารการเงิน
      1. ตราสารการเงินในตลาดเงิน (money market instruments)
      2. ตราสารหนี้ระยะยาว (long-term debt instruments)
      3. ตราสารทุน (equity security)
      4. ตราสารอนุพันธ์ (derivative security)
    2. กระบวนการตัดสินใจลงทุน
      1. ขั้นตอนการตัดสินใจลงทุน
      2. การกระจายความเสี่ยงกับการลงทุน
      3. บทบาทของผู้แนะนำการลงทุน
    3. บทสรุป

  • บทที่ 3 อัตราผลตอบแทนและความเสี่ยงจากการลงทุนในหลักทรัพย์

    1. อัตราผลตอบแทนจากการลงทุนในหลักทรัพย์
      1. องค์ประกอบของอัตราผลตอบแทนจากการลงทุนในหลักทรัพย์
      2. อัตราผลตอบแทนในช่วงถือครอง (holding period return: HPR)
      3. อัตราผลตอบแทนเฉลี่ย (average rate of return)
      4. อัตราผลตอบแทนที่คาดหวัง (expected rate of return)
      5. อัตราผลตอบแทนที่เป็นตัวเงินและอัตราผลตอบแทนที่แท้จริง
      6. อัตราผลตอบแทนหลังภาษี (after-tax rate of return)
    2. แนวคิดมูลค่าเงินตามเวลาของเงิน (time value of money: TVM)
      1. การคำนวณหามูลค่าปัจจุบันและมูลค่าอนาคตของเงินจำนวนเดียว
      2. การคำนวณหามูลค่าปัจจุบันและมูลค่าอนาคตของกระแสเงินสดหลายจำนวน
      3. อัตราผลตอบแทนทบค่า (effective annual rate: EAR)
    3. ความเสี่ยงจากการลงทุนในหลักทรัพย์
      1. ประเภทของความเสี่ยงจากการลงทุนในหลักทรัพย์
      2. การวัดความเสี่ยงจากการลงทุนในหลักทรัพย์ (measurement of investment risk)
      3. ความสัมพันธ์ระหว่างอัตราผลตอบแทนและความเสี่ยง จากการลงทุนในหลักทรัพย์
      4. เปรียบเทียบอัตราผลตอบแทนและความเสี่ยงจากการลงทุน ในตราสารทุน ตราสารหนี้ และตราสารตลาดเงิน
    4. บทสรุป

  • บทที่ 4 อัตราผลตอบแทนและความเสี่ยงของกลุ่มหลักทรัพย์

      1. ความหมายและองค์ประกอบของกลุ่มหลักทรัพย์
      2. การคำนวณอัตราผลตอบแทนที่คาดหวังของกลุ่มหลักทรัพย์
    2. การวัดความเสี่ยงของกลุ่มหลักทรัพย์
      1. ความแปรปรวนร่วม (covariance) และสัมประสิทธิ์สหสัมพันธ์ (correlation)
      2. ค่าความแปรปรวนและส่วนเบี่ยงเบนมาตรฐานของอัตราผลตอบแทน ของกลุ่มหลักทรัพย์
      3. ประเภทของความเสี่ยงในมุมมองของการกระจายการลงทุน
    3. ความสัมพันธ์ระหว่างอัตราผลตอบแทนและความเสี่ยงตาม CAPM (capital asset pricing model)
      1. ค่าเบต้า
      2. ความสัมพันธ์ระหว่างอัตราผลตอบแทนที่ผู้ลงทุนต้องการกับค่าเบต้า
      3. ค่าเบต้าของกลุ่มหลักทรัพย์ (portfolio beta)
    4. การประเมินผลการบริหารกลุ่มหลักทรัพย์ด้วยอัตราผลตอบแทน ที่ปรับค่าความเสี่ยง
      1. Sharpe Model
      2. Treynor Model
      3. Jensen Model
    5. บทสรุป

  • บทที่ 5 การวิเคราะห์เศรษฐกิจและอุตสาหกรรม

    1. แนวคิดพื้นฐานในการวิเคราะห์เศรษฐกิจ
      1. เครื่องชี้วัดการบรรลุเป้าหมายทางเศรษฐกิจและการดำเนินนโยบายเศรษฐกิจ
      2. ตัวเลขทางเศรษฐกิจที่สำคัญ
      3. วัฏจักรเศรษฐกิจ
      4. การพยากรณ์เศรษฐกิจ
    2. การวิเคราะห์อุตสาหกรรม
      1. กระบวนการวิเคราะห์อุตสาหกรรม
      2. การจำแนกประเภทอุตสาหกรรม
      3. ความสัมพันธ์ระหว่างวัฏจักรธุรกิจและอุตสาหกรรม
      4. ตัวแปรสำคัญที่มีผลกระทบต่ออุตสาหกรรม
      5. การวิเคราะห์วัฏจักรชีวิตของอุตสาหกรรม
      6. โครงสร้างการแข่งขันในอุตสาหกรรม
      7. การวิเคราะห์ภาวะการแข่งขันในอุตสาหกรรม
      8. แหล่งข้อมูลเพื่อการวิเคราะห์อุตสาหกรรม
    3. บทสรุป

  • บทที่ 6 การวิเคราะห์บริษัท

    1. แนวคิดพื้นฐานของการวิเคราะห์บริษัท
      1. ความหมายและความสำคัญของการวิเคราะห์บริษัท
      2. แนวคิดและเครื่องมือในการวิเคราะห์บริษัทในเชิงคุณภาพ
    2. การวิเคราะห์งบการเงิน
      1. ความหมายและความสำคัญของการวิเคราะห์งบการเงิน
      2. การวิเคราะห์ common size และแนวโน้ม
      3. อัตราส่วนทางการเงินเพื่อการวิเคราะห์งบการเงิน
      4. การประยุกต์การวิเคราะห์งบการเงินและข้อจำกัด ของการใช้อัตราส่วนทางการเงิน
      5. การวิเคราะห์และการติดตามความสามารถในการประกอบธุรกิจ
    3. การลงทุนที่คำนึงถึงความยั่งยืน
      1. ความหมายและความสำคัญของการลงทุนที่คำนึงถึงความยั่งยืน
      2. แนวคิดและเครื่องมือในการลงทุนที่คำนึงถึงความยั่งยืน
    4. บทสรุป

  • บทที่ 7 การวิเคราะห์หลักทรัพย์โดยใช้ปัจจัยทางเทคนิค

    1. แนวคิดพื้นฐานของการวิเคราะห์หลักทรัพย์โดยใช้ปัจจัยทางเทคนิค
      1. ความหมายและความสำคัญของการวิเคราะห์หลักทรัพย์ทางเทคนิค
      2. ลักษณะของข้อมูลที่นำมาวิเคราะห์ทางเทคนิค
      3. รูปแบบการเปลี่ยนแปลงของราคาหลักทรัพย์
      4. ตัวอย่างการประยุกต์แนวรับและแนวต้านเพื่อการตัดสินใจลงทุน
    2. บทสรุป

  • บทที่ 8 การลงทุนในหลักทรัพย์ต่างประเทศ

    1. เหตุผลที่พิจารณาหลักทรัพย์ต่างประเทศเป็นส่วนหนึ่งของกลุ่มหลักทรัพย์
      1. เป็นการกระจายการลงทุน (diversification) โดยมุ่งหวังลดความเสี่ยง
      2. เป็นช่องทางหนึ่งเพื่อมุ่งหวังสร้างความเติบโตของมูลค่ากลุ่มหลักทรัพย์
      3. เป็นทางเลือกในการลงทุนในธุรกิจหรืออุตสาหกรรมที่น่าสนใจ แต่มิได้เป็นหลักทรัพย์จดทะเบียนในไทย
    2. ช่องทางของการลงทุนและผู้ที่เกี่ยวข้องกับการลงทุนในหลักทรัพย์ต่างประเทศ
      1. ช่องทางของการลงทุนในหลักทรัพย์ต่างประเทศ
      2. ผู้ที่เกี่ยวข้องกับการลงทุนในหลักทรัพย์ต่างประเทศ
    3. ผลตอบแทนและความเสี่ยงของการลงทุนในหลักทรัพย์ต่างประเทศ
      1. การคำนวณอัตราผลตอบแทนในช่วงถือครอง (holding period return: HPR) ของการลงทุนในหลักทรัพย์ต่างประเทศ
      2. ความเสี่ยงของการลงทุนในหลักทรัพย์ต่างประเทศ
    4. อัตราแลกเปลี่ยนเงินตราต่างประเทศและปัจจัยกำหนด การเปลี่ยนแปลงอัตราแลกเปลี่ยน
      1. อัตราแลกเปลี่ยนเงินตราต่างประเทศ
      2. ตลาดเงินตราต่างประเทศ
      3. การตีความการเปลี่ยนแปลงค่าของอัตราแลกเปลี่ยน
      4. การบริหารความเสี่ยงจากอัตราแลกเปลี่ยนเงินตราต่างประเทศ
    5. ประเด็นอื่น ๆ ที่ต้องคำนึงถึงในการลงทุนหลักทรัพย์ต่างประเทศ
      1. การจัดสรรวงเงินการซื้อขายตราสารสกุลเงินตราต่างประเทศ
      2. หลักทรัพย์ที่อนุญาตให้ลงทุนได้
      3. ค่าใช้จ่ายต่างๆ
      4. ความแตกต่างของเวลา เกณฑ์การซื้อขาย และการชำระราคา
      5. ช่องทางการติดตามข่าวสารและราคาหุ้นต่างประเทศ
    6. บทสรุป
